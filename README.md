# Life Compass - あなたと大切な人の人生の羅針盤

MBTI、数秘術（ともこ式）、西洋占星術を統合した高品質な人生相談サイト

## 概要

Life Compassは、3つの診断手法を組み合わせて、多角的な視点から自己理解と人間関係の改善をサポートします。

### 主要機能

- **個人診断**: MBTI、数秘術、西洋占星術による自己分析
- **相性診断**: 2人の相性を多角的に分析
- **保存機能**: 診断結果を保存して再利用可能

## 技術スタック

- React 18
- Vite 6
- Tailwind CSS 4
- Framer Motion
- shadcn/ui
- Lucide React

## セットアップ

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm run dev

# ビルド
pnpm run build
```
